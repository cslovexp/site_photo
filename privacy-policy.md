# 개인정보 처리방침 초안

현장사진대지는 사용자의 공사 현장 사진과 보고서 데이터를 기기 안에서 처리하는 로컬 우선 앱입니다.

## 수집하는 정보

앱은 MVP 기준으로 서버 계정, 로그인 정보, 위치 정보, 분석 이벤트를 수집하지 않습니다.

사용자가 직접 입력하거나 생성하는 데이터:

- 현장명
- 현장 주소
- 발주처/고객명
- 업체명
- 담당자명
- 현장사진
- 사진 메모, 공종, 위치, 시공 단계
- 생성된 PDF 보고서

위 데이터는 사용자의 기기 내 로컬 저장소에 저장됩니다.

## 사진 및 파일 처리

사용자가 카메라로 촬영하거나 앨범에서 선택한 사진은 앱 문서 디렉터리로 복사되어 저장됩니다. 앱은 임시 사진 경로에 의존하지 않으며, 저장된 파일 경로만 로컬 데이터베이스에 기록합니다.

생성된 PDF 보고서는 앱 문서 디렉터리의 보고서 저장 영역에 저장됩니다.

## 권한 사용 목적

- 카메라: 현장사진 촬영
- 사진 보관함: 앨범에서 현장사진 선택
- 파일/공유 기능: 생성한 PDF 보고서 공유

권한은 위 기능을 수행하기 위해서만 사용됩니다.

## 외부 전송

MVP 기준으로 앱은 사진, 현장 정보, PDF 보고서를 자체 서버로 업로드하지 않습니다. 사용자가 공유 기능을 직접 실행한 경우에만 선택한 공유 대상 앱으로 PDF 파일이 전달됩니다.

## 결제 정보

Pro 구독 또는 구매 기능은 App Store 또는 Google Play의 결제 시스템을 통해 처리됩니다. 앱은 카드번호 등 결제 수단 정보를 직접 수집하거나 저장하지 않습니다.

## 데이터 삭제

사용자는 기기에서 앱을 삭제하여 앱에 저장된 로컬 데이터를 삭제할 수 있습니다. 향후 앱 내부 데이터 삭제 기능이 추가될 수 있습니다.

## 문의

개인정보 관련 문의는 앱 배포자가 제공하는 고객지원 이메일로 연락할 수 있습니다.
